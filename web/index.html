RDMA memory registration

RdmaTransport: Failed to register memory; common causes are memlock not being unlimited and nvidia_peermem not being loaded.

Check shell

ulimit -l
lsmod | grep nvidia_peermem || true

Repair shell

ulimit -l unlimited
sudo modprobe nvidia_peermem

NVIDIA container runtime hook

docker: Error response from daemon: exec: "nvidia-container-runtime-hook": executable file not found in $PATH.

Check shell

which nvidia-container-runtime-hook || true
nvidia-ctk --version
sudo docker info | grep -i 'runtime\|nvidia' || true

Repair shell

sudo nvidia-ctk runtime configure --runtime=docker
sudo systemctl restart docker

RoCE bond / netplan

Check shell

ls /etc/netplan
show_gids

Repair shell

sudo rm -i /etc/netplan/01-netcfg.yaml
sudo netplan apply
